# Hundred Xdi8

由荊哲設計，Nyoeghau (Edward) 補充字形字重和字體化的希頂語字體。有可變 `ttf` 字體 (variable font) 與普通 `otf` 兩種格式供選擇。

`Opentype` 特性：

1. `ss01` 大寫 (alias: `case`)
2. `ss02` 中寫
3. `ss03` 數字

提供字重：12, 15, 18, 21, 24, 27, 30, 33, 36, 39, 42, 45, 48, 51, 54, 57, 60, 63, 66, 69, 72。可變字體另可自定義字重。
